## En non-profit erhvervsdrivende fond

**Mind over Machine** er en non-profit erhvervsdrivende fond. En kombineret tænketank og laboratorium.

Med udgangspunkt i langsigtet systemtænkning udforsker og afprøver vi principper og teknikker, som kan gøre software til et regenerativt redskab.

I Laboratoriet udvikler vi konkrete Open Source software-systemer for vores medlemmer og kunder, som er baseret på disse principper og teknikker. Vi indhenter feedback og justerer løbende vores antagelser, principper og teknikker, som vi deler åbent.

### Vores mission

At afsøge og udbrede en ny model for hvordan software kan bidrage til et regenerativt paradigme, som samlet bidrager _positivt_ til mennesker og Verden.
