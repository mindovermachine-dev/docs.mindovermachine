# Stiftelsespakke checkliste (intern)

Denne checkliste er intern arbejdsdokumentation til stiftelse og registrering af en erhvervsdrivende fond.

Kildegrundlag: Erhvervsstyrelsens vejledning om stiftelse af en erhvervsdrivende fond.

## 1. Vedtægt og grunddata

- [ ] Vedtægten er dateret og underskrevet af stifter(e).
- [ ] Vedtægten indeholder fondens navn og eventuelle binavne.
- [ ] Ordet "fond" indgår i navn/binavn.
- [x] Stifter(e) er oplyst i vedtægten.
- [ ] Formål er præcist beskrevet, herunder hvordan overskud anvendes.
- [ ] Grundkapitalens størrelse fremgår tydeligt.
- [ ] Det fremgår hvordan kapitalen er indbetalt (kontant eller andre værdier).
- [ ] Det fremgår om fonden overtager andre værdier end kontanter ved stiftelsen.
- [ ] Eventuelle særlige rettigheder/fordele til stiftere eller andre er beskrevet.
- [ ] Antal bestyrelsesmedlemmer og udpegningsmåde er beskrevet.
- [ ] Regnskabsår og første regnskabsår fremgår.
- [ ] Eventuel dato for senere retsvirkning fremgår tydeligt.

## 2. Bilag til vedtægten

- [ ] Alle dokumenter, som vedtægten henviser til, er samlet og klar til indsendelse.
- [ ] Forretningsorden er vedlagt som bilag.
- [ ] Instrukser er vedlagt som bilag.
- [ ] Eventuelle øvrige stiftelsesdokumenter (gavebrev, testamente eller lignende) er vedlagt.
- [ ] Vedtægt og bilag er samlet i et format, der kan offentliggøres ved registrering.

## 3. Kapital og dokumentation

- [ ] Grundkapitalen opfylder minimumskravet på 300.000 kr.
- [ ] Kapitalen er fuldt indbetalt før anmeldelse.
- [ ] Kapitalgodkendelse er klar via bank, advokat eller godkendt revisor.
- [ ] Ved apportindskud foreligger vurderingsberetning eller gyldig ledelseserklæring.
- [ ] Ved indskud af bestående virksomhed foreligger åbningsbalance med revisorerklæring.

## 4. Erhvervsdrift og økonomisk grundlag

- [ ] Der er udarbejdet en kort, konkret beskrivelse af fondens erhvervsdrift.
- [ ] Det er beskrevet i hvilket omfang driften sker på markedsmæssige vilkår.
- [ ] Der foreligger et overslag for 3-5 års forventede erhvervsmæssige bruttoindtægter.
- [ ] Der foreligger eventuelt en simpel balance over forventede aktiver og passiver.
- [ ] Eventuelle offentlige tilskud og andet offentligt tilsyn er beskrevet.

## 5. Ledelse, revisor og registreringsoplysninger

- [ ] Bestyrelsen er valgt før registrering.
- [ ] Revisor er valgt før registrering.
- [ ] Tegningsregel er fastlagt og registreringsklar.
- [ ] Navn, funktion, adresse og identitetsoplysninger for ledelsen er indsamlet korrekt.
- [ ] Oplysninger om reelle ejere er afklaret og registreringsklare.

## 6. Udenlandske stiftere/personer (kun hvis relevant)

- [ ] Registreringsbevis og registreringsnummer for udenlandsk virksomhed er vedlagt.
- [ ] Pas/ID-oplysninger for personer uden CPR er indhentet og dokumenteret.
- [ ] Nødvendig dokumentation for handle- og retsevne er indhentet.

## 7. Tidsfrister og indsendelse

- [ ] Anmeldelse planlagt senest 2 uger efter stiftelsesbeslutning.
- [ ] Kontrol af at fonden ikke handler ud over tilladt ramme før registrering.
- [ ] Alt materiale er kvalitetssikret før indsendelse på Virk.
- [ ] Ansvarlig person for indsendelse er udpeget.
- [ ] Intern deadline er sat mindst 2 arbejdsdage før officiel frist.

## 8. Intern slutkontrol

- [ ] Vedtægtstekst stemmer med registreringsoplysninger.
- [ ] Bilag er versionslåst (dato og versionsnummer).
- [ ] Der er lavet intern gennemgang af juridiske risikopunkter.
- [ ] Der er taget backup af hele indsendelsespakken.
- [ ] Kvittering for anmeldelse gemmes centralt efter indsendelse.
