# Ways You Can Contribute

## Leave comments on the public page

The content is publicly available at https://docs.mindovermachine.dk all pages have giscus enabled, you can log in with your GitHub account — first time, you're asked to authorize the giscus app on your account — then you can leave comments.

## You are more than welcome to raise issues to this repo:

On Github, simply create an issue: [`mindovermachine-dev/docs.mindovermachine/issues`](https://github.com/mindovermachine-dev/docs.mindovermachine/issues/new/choose)

## Hack your own changes

Requires Git, VS Code and a Docker engine (like OrbStack or Docker Desktop) installed on your PC.

_Note: If you are on Windows, running from WSL is recommended"_

**With write access**

- Clone the repo
- Open it in VS Code
- Accept the offer to start the DevContainer
- Accept the offer to work in the default workspace
- Work the `workon`, `wrapup` and `deliver` commands documented in [TakT](https://github.com/devx-cafe/gh-tt)

**Without write access**

😜 Obtain write access! (reach out to @lakruzz) It's version controlled, you can not break anything!

Alternatively:

- Create a personal fork
- Work anyway you want
- Create Pull Request
